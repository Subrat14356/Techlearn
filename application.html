Application Software

Application software consists of programs designed for specific user tasks, such as word processing, accounting, and multimedia editing.

Categories of Application Software:

Word Processing

Create, edit, format, and print documents

Examples:

  • Microsoft Word
  • Google Docs
  • LibreOffice Writer

Accounting Software

Manage financial transactions and records

Examples:

  • Tally
  • QuickBooks
  • Xero

Desktop Publishing

Create professional-quality publications

Examples:

  • Adobe InDesign
  • CorelDRAW
  • Adobe Photoshop

Media Players

Play audio and video files

Examples:

  • VLC Media Player
  • Windows Media Player
  • iTunes

Web Browsers

Access and navigate the World Wide Web

Examples:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

Database Software

Organize, store, and retrieve data

Examples:

  • Microsoft Access
  • Oracle
  • MySQL

Types of Operating Systems

Single-User, Single-Tasking OS

Allows only one user to run one program at a time

Examples:

  • MS-DOS
  • Palm OS (for PDAs)

Multi-User, Multi-Tasking OS

Allows multiple users to run multiple programs simultaneously

Examples:

  • Windows (7, 10, 11)
  • macOS
  • Linux distributions

Real-Time OS (RTOS)

Designed for systems that require immediate response to input signals

Examples:

  • VxWorks
  • QNX
  • FreeRTOS

Applications:

  • Medical devices
  • Industrial robots
  • Automotive systems

Mobile OS

Designed specifically for smartphones and tablets

Examples:

  • Android
  • iOS
  • HarmonyOS
