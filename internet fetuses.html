Internet Access

Internet access is the process of connecting devices to the internet.

Types of Internet Connections:

  • Dial-up Connection
    • Transmits analog signals over an ordinary telephone line.
    • Requires dialing a local ISP (Internet Service Provider).
    • Only one service can be used at a time (either internet or phone call).
    • Only one computer can be connected.
  • Direct Internet Connection
    • Allows a computer to send and receive internet data packets.
    • Requires special software that supports the TCP/IP protocol.
  • Broadband Connection
    • Provides high-speed internet access.
    • Supports multiple devices simultaneously.

Direct Internet Access Processes

Direct Internet Access is Achieved by Two Processes:

  1. Dial-up Connection (TP)
    • Uses a modem and telephone lines for internet access.
    • Special software supporting SLIP (Serial Line Internet Protocol) or PPP (Point-to-Point Protocol) is required.
    • SLIP: Serial Line Internet Protocol
    • PPP: Point-to-Point Protocol
  2. LAN Connection
    • If a computer is connected to the LAN, it is directly linked to the network.

Broadband Connection (LQ)

  • Refers to high-speed internet technology that transmits data at very high speeds.
  • Uses cables or mobile phone networks for communication.
  • Works by using a wide range of frequencies divided into channels, allowing internet data and telephone calls to happen simultaneously.

DSL (Digital Subscriber Line)

  • It is a wired technology.
  • Data is transmitted using ordinary telephone lines.
  • Allows simultaneous transmission of internet data and voice data.
  • Provides data transmission speeds ranging from Kbps to Mbps.

Internet Speed and Availability

The internet speed and availability depend on the distance from the service provider.

Types of DSL Technology

  1. ADSL (Asymmetrical Digital Subscriber Line)
    • Commonly used by home users and businesses.
    • Offers higher downstream (download) speeds compared to upstream speeds.
    • Allows simultaneous telephone calls and internet service without interruption.
  2. SDSL (Symmetrical Digital Subscriber Line)
    • Used by businesses and organizations.
    • Provides equal speeds for both downstream (download) and upstream (upload).
    • Often used for video conferencing and other data-intensive applications.

Cable Modem

  • A cable modem is an external device used for internet access.
  • It is provided by cable operators to deliver internet services.

Cable Modem Features

  • Uses a single cable for telephone and internet services.
  • Offers data transmission speeds up to 1.5 Mbps or more.
  • Multiple computers can be connected, but more users may slow the connection.
  • Supports simultaneous transmission of internet and TV data.

Optical Fiber Cable

  • Converts signals into light for high-speed data transmission.
  • Can transmit data across extremely thin fiber strands.
  • Provides higher downstream than upstream speeds.
  • Faster than DSL or cable modem connections.
  • Allows simultaneous voice and internet data transmission.

Broadband Technologies

Integrated Services Digital Network (ISDN)

  • An older technology that provides high-speed internet connection.
  • Uses an ordinary telephone line to connect to the internet.
  • Can transmit both voice and internet data simultaneously.
  • Allows multiple computers to be connected via a single line.

Wireless Broadband

  • Connects home users or businesses via radio signals between the customer and service provider.
  • Can be either mobile or fixed.
  • Ideal for remote areas where DSL or cable modem services are unavailable.
  • Requires an antenna and modem to establish a connection.
  • Provides speeds comparable to DSL or cable modem.
  • Companies providing wireless broadband are known as Wireless Internet Service Providers (WISPs).
  • Typically offers speeds around 2 Mbps to unlicensed devices.
Wireless Local Area Network
  • Provides seamless broadband access over short distances.
  • Uses Wi-Fi networks with unlicensed devices.
  • Designed for internet access in homes or businesses.

Wireless Connectivity

Mobile Wireless
  • Mobile broadband services are provided by telecom service providers.
  • Primarily uses 3G technology, including:
    • HSDPA: Enables high-speed downloading.
    • HSUPA: Facilitates high-speed uploading.
  • Speeds range from 100 kbps to several Mbps.

Satellite Internet

  • Satellite technology provides connectivity for telephone and television services.
  • It can also offer broadband connections, especially in remote areas.

BPL (Broadband over Power Line)

  • BPL delivers broadband using low and medium-voltage electric lines.

Internet Service Providers (ISP)

  • An ISP is a private company or government entity that provides internet services to individuals and organizations.

Network Hardware Components

Network Interface Card (NIC)

  • Also known as LAN Card or Ethernet Interface Card.
  • Provides a physical connection to the network.
  • Attaches the MAC address to data packets.

HUB in Networking

  • A HUB is a communication device that acts as a central point in a star topology network.
  • Receives and forwards data.
  • Available in configurations of 4, 8, 12, or 24 ports.
  • Uses RJ-45 cables.

Advantages of a HUB

  • ✔ Low Cost – Affordable and easy to install.
  • ✔ Connects Multiple Devices.

Disadvantages of a HUB

  • ❌ Extends Collision Domain.
  • ❌ Does Not Filter Data.

Switch - A Network or Communication Device

  • A switch selects the port or circuit to send data packets to the next network segment.
  • It functions like a router.
  • It is an intelligent device.
  • It contains a bridging table to track hardware addresses.
  • It tracks MAC addresses of all connected devices.
  • It operates at the Data Link Layer of the OSI model.
  • Each port on a switch is a collision domain.

Advantages of a Switch:

  • Limits the collision domain.
  • Filters data packets.
  • Keeps track of hardware addresses using a bridging table.
  • Can be configured to limit the broadcast domain.

Disadvantages of a Switch:

  • More expensive than a hub.
  • Configuration is complex.

Difference Between Hub and Switch

Feature Hub Switch
Layer Physical Layer Data Link Layer

Repeater - A Network Device

  • A Repeater is a network device used to amplify and regenerate weak signals.
  • It helps in creating long-distance networking.
  • It operates at the Physical Layer of the OSI model.
  • A multi-port repeater is known as a Hub.

Basic Functions of a Repeater:

  • Receives weak signals.
  • Regenerates the signals to avoid collisions.
  • Transmits the signal to the next network segment.

Advantages of a Repeater:

  • Can connect different types of media (cables).
  • Extends a network in terms of distance.
  • Does not introduce additional network traffic.

Disadvantages of a Repeater:

  • Extends the collision domain.
  • Does not filter data packets.
  • Limits the number of repeaters in a network.

Bridge - A Network Device

  • A Bridge is a network device used to connect LANs that use the same protocol.
  • It operates at the Data Link Layer of the OSI model.
  • Uses a bridging table to track devices on each segment.

Filtering and Spanning Tree Protocol (STP)

  • Filters data packets and determines whether to forward them to the next destination.
  • Uses Spanning Tree Protocol (STP) to examine data packets and prevent network loops.

Advantages:

  • Limits the collision domain.
  • Can extend network distance.
  • Uses MAC addresses to filter traffic.
  • Can connect different types of media.

Disadvantages:

  • Works only with the same type of protocol.
  • Broadcast packets cannot be filtered.
  • More expensive than a repeater.
  • Slower processing than a repeater due to additional packet processing.

Router - A Network Device

  • A Router is a network device used to find the best path to send data packets to the next network segment.
  • Operates at the Network Layer of the OSI model.
  • Works in LAN, MAN, and WAN environments.
  • Used to connect different types of computer networks.
  • Functions like a firewall.
  • Can change packet routes based on the network segment.
  • Maintains routing tables to select the best path.

Router Features

  • Contains a routing table to keep track of network routes.
  • Static Routing is manually configured by the network administrator.
  • Dynamic Routing updates automatically when network topology changes.
  • Uses routing protocols to find the best path using special algorithms.
  • Helps in resource access by selecting the optimal path.

Advantages:

  • Limits the collision domain.
  • Determines the best path for data transmission.
  • Filters broadcast messages.
  • Works in any environment (LAN, MAN, WAN).
  • Can connect different types of computer networks.

Disadvantages:

  • More expensive than other network devices.
  • Requires the use of routable protocols.
  • Difficult to configure.

Gateway - A Network Device

  • A Gateway is an entry point to another network.
  • Functions as a host computer at the starting and ending points of a network.
  • Operates at all layers of the OSI model.
  • Used to connect heterogeneous networks.

Types of Gateways

There are different types of gateways used to connect networks, such as:

  1. Address Gateway
    • Connects networks using the same protocol but with a different directory space.
    • Example: Message handling service that connects different email systems.
  2. Protocol Gateway
    • Connects networks using different protocols.
    • Translates secure protocols so the destination can understand them.
  3. Application Gateway
    • Translates applications such as an internet email server to a message server.
