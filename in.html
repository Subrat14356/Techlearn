Computer Input Devices

Learn about the essential hardware components that allow users to interact with computers

Input Unit

The input unit consists of devices that allow users to send data, commands, and instructions to the computer system.

Common Examples:
  1. Keyboard - Primary text input device
  2. Mouse - Pointing and selection device
  3. Joystick - Gaming and control device
  4. OMR - Optical Mark Recognition for forms
  5. OCR - Optical Character Recognition for text

Keyboard

The standard keyboard contains 104 to 106 keys organized into functional groups:

Special Keys

These perform specific system functions and navigation:

Examples: Esc, Tab, Caps Lock, Shift, Ctrl, Alt, Windows, Enter, Spacebar, Backspace, Delete, Insert, Home, End, Page Up, Page Down, Print Screen, Scroll Lock, Pause

Function Keys

12 programmable keys (F1-F12) with software-specific functions:

  • F1 - Universal help key
  • F5 - Refresh in browsers
  • F11 - Fullscreen mode

Typewriter Keys

The alphanumeric section resembling traditional typewriters:

A-Z letters, 0-9 numbers, and punctuation marks arranged in QWERTY layout

Mouse

The mouse is a pointing device that detects two-dimensional motion relative to a surface.

Evolution of Mouse Technology

  1. Mechanical Mouse (1960s) - Rubber ball and rollers
  2. Optical Mouse (1999) - LED light and sensor
  3. Laser Mouse (2004) - More precise than optical
  4. Bluetooth Mouse - Wireless connectivity
  5. Touch Surface Mouse - No physical buttons

Connection Types

  • Serial - 9-pin connector (obsolete)
  • PS/2 - Round 6-pin connector
  • USB - Universal standard (Type A)
  • Wireless - RF or Bluetooth
