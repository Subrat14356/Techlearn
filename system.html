System Software

System Software

System software manages hardware resources and provides a platform for application software. It acts as an interface between the computer and its hardware.

Examples of System Software:

  • Windows (11, 10, 8, 7)
  • Linux (Ubuntu, Fedora, Debian)
  • Unix (macOS, Solaris)
  • Android
  • iOS

Types of System Software

1. Operating System (OS)

The OS manages computer resources, provides a user interface, and runs applications.

Examples:

  • Windows 10/11
  • macOS
  • Android
  • Linux distributions

Key Functions:

  • Process management
  • Memory management
  • File system management
  • Device management
  • User interface

2. Network Operating System (NOS)

Used to manage and control network operations in multi-user environments.

Examples:

  • Windows Server
  • Linux Server
  • Novell NetWare

Features:

  • User authentication
  • File sharing
  • Printer sharing
  • Network security

3. Utility Software

Programs that help maintain and optimize a computer system.

Categories:

  • Antivirus Software: Norton, McAfee, Windows Defender
  • File Management Tools: File Explorer, Finder
  • Disk Tools: Cleanup, Defragmentation, Formatting
  • Backup Software: Time Machine, Windows Backup

4. Device Drivers

Programs that allow communication between hardware devices and the operating system.

Examples:

  • Printer drivers
  • Graphics card drivers
  • Network adapter drivers
  • Audio drivers

Types of Operating Systems

Single-User, Single-Tasking OS

Allows only one user to run one program at a time

Examples:

  • MS-DOS
  • Palm OS (for PDAs)

Multi-User, Multi-Tasking OS

Allows multiple users to run multiple programs simultaneously

Examples:

  • Windows (7, 10, 11)
  • macOS
  • Linux distributions

Real-Time OS (RTOS)

Designed for systems that require immediate response to input signals

Examples:

  • VxWorks
  • QNX
  • FreeRTOS

Applications:

  • Medical devices
  • Industrial robots
  • Automotive systems

Mobile OS

Designed specifically for smartphones and tablets

Examples:

  • Android
  • iOS
  • HarmonyOS
