Memory Units and Storage Devices

Bit (Binary Digit)

A bit (0 or 1) is the smallest unit of data in a computer. All data in computers is represented using bits.

Developed By: Lady Ada Lovelace - Often regarded as the first computer programmer for her work on Charles Babbage's Analytical Engine.

Types of Memory

Main Memory (Primary Memory)

  • ROM (Read-Only Memory)
  • RAM (Random Access Memory)
  • PROM (Programmable Read-Only Memory)
  • SRAM (Static RAM)
  • DRAM (Dynamic RAM)
  • EROM (Erasable Read-Only Memory)
  • EAROM (Electrically Alterable Read-Only Memory)

Mass Memory (Secondary Memory)

  • Magnetic Tape
  • Magnetic Disk
  • Optical Disk
  • Solid State Drives (SSD)
  • Flash Memory
  • Cloud Storage

Difference Between Main Memory and Mass Memory

Main Memory Mass Memory
Located inside the computer Located outside the computer
Faster access speed Slower access speed
Stores temporary data Stores permanent data
Volatile (RAM loses data when power is off) Non-volatile (retains data even when power is off)
More expensive per GB Less expensive per GB

Types of Main Memory

ROM (Read-Only Memory)

As the name suggests, it is read-only and cannot be modified after manufacturing. The user can only read the stored information but cannot write or modify it.

Types of ROM:

  • PROM (Programmable Read-Only Memory): Can be programmed once after manufacturing.
  • EPROM (Erasable Programmable Read-Only Memory): Can be erased and reprogrammed using UV light.
  • EAROM (Electrically Alterable Read-Only Memory): Can be modified electrically but at a slower rate.

RAM (Random Access Memory)

Used for temporary storage while the computer is running. Data is lost when power is turned off (volatile memory). RAM allows both reading and writing of data.

Types of RAM:

  • SRAM (Static RAM): Faster and does not need refreshing, used in cache memory.
  • DRAM (Dynamic RAM): Slower and requires constant refreshing, used in main memory.
  • SDRAM (Synchronous DRAM): Synchronized with the system clock for better performance.
  • DDR SDRAM (Double Data Rate SDRAM): Transfers data on both rising and falling edges of the clock signal.

Storage Devices

Magnetic Tape

A secondary storage device used for storing large amounts of data. Made of magnetic oxide with a thin plastic coating.

Function of Magnetic Tape:

  • Functions similarly to an audio cassette
  • Consists of two spools with a magnetic reel wound between them
  • Uses read/write heads to access information
  • Primarily used for backups and archival storage

Magnetic Disk

A secondary storage device used for data storage. Divided into tracks and sectors for organizing data.

Storage Organization:

  • Track: Circular path on the disk surface
  • Sector: Smallest storage unit (typically 512 bytes)
  • Cluster: Group of sectors treated as a single unit

Floppy Disk

A secondary storage device used for storing small amounts of data. Made of a magnetic oxide layer coated on thin plastic film, enclosed in a protective cover.

Floppy Disk Storage Conditions:

  • Must be stored at temperatures between 10°C to 60°C
  • Should be kept away from direct sunlight
  • Must be protected from magnets to avoid data loss
  • Handle carefully to prevent physical damage

Floppy Disk Specifications:

Size Number of Tracks Number of Sectors Storage Capacity
3.5 inch 80 18 1.44 MB
5.25 inch 80 15 1.2 MB

Hard Disk

A secondary storage device used to store large amounts of data. Consists of multiple disks combined to form a disk pack.

Components of a Hard Disk:

  • Platters (disks coated with magnetic material)
  • Spindle (rotates the platters)
  • Read/write heads (one per platter surface)
  • Actuator arm (moves the heads)
  • Controller board (manages operations)

Types of Hard Disks:

Internal Hard Disk

  • Installed inside the system unit
  • Used for storing operating systems and applications
  • Higher performance than external drives
  • Available in various capacities (500GB to 10TB+)

External Hard Disk

  • Portable and connects via USB or other interfaces
  • Used for additional storage and backups
  • Available in HDD and SSD formats
  • Typically slower than internal drives

Optical Disk

A storage medium that uses laser technology to read and write data. Commonly used for multimedia applications and software distribution.

Functionality:

  • Uses laser beam to read/write data on reflective surface
  • Data stored as pits (indentations) and lands (flat areas)
  • Different wavelengths for different formats (CD, DVD, Blu-ray)

Types of Optical Disks:

CD (Compact Disc)

  • Capacity: 700MB
  • Wavelength: 780nm (infrared)
  • Used for audio, data, and small applications

DVD (Digital Versatile Disc)

  • Capacity: 4.7GB (single-layer) or 8.5GB (dual-layer)
  • Wavelength: 650nm (red)
  • Used for video, software, and larger data storage

Blu-ray Disc

  • Capacity: 25GB (single-layer) or 50GB (dual-layer)
  • Wavelength: 405nm (blue-violet)
  • Used for HD video and large data storage

Solid State Drives (SSD)

SSDs use flash memory to store data, offering faster performance and higher durability than traditional hard disks.

Key Features:

  • No moving parts (more resistant to physical shock)
  • Faster read/write speeds (especially for random access)
  • Lower power consumption
  • More expensive per GB than HDDs
  • Limited write cycles (though modern SSDs last many years)

Applications:

  • Operating system drives for faster boot times
  • High-performance computing applications
  • Laptops and mobile devices for better battery life
  • Gaming systems for faster level loading

Flash Memory and USB Drives

Flash Memory Cards

Solid-state storage devices commonly used in portable devices like cameras and smartphones.

Types:

  • SD (Secure Digital) cards
  • microSD cards
  • CompactFlash
  • Memory Stick

USB Drives

Portable storage devices that connect via USB ports, also known as flash drives or thumb drives.

Key Features:

  • Compact and lightweight
  • Storage capacities from 16GB to 2TB
  • Plug-and-play functionality
  • No power source needed
  • Used for data transfer, backups, and system recovery

Cloud Storage

Cloud storage allows users to store data on remote servers accessed via the internet, rather than on local storage devices.

Types of Cloud Services:

IaaS (Infrastructure as a Service)

Provides virtualized computing resources over the internet (e.g., AWS, Azure, Google Cloud)

PaaS (Platform as a Service)

Provides platforms allowing customers to develop, run, and manage applications (e.g., Heroku, Google App Engine)

SaaS (Software as a Service)

Delivers software applications over the internet (e.g., Google Workspace, Microsoft 365)

Advantages of Cloud Storage:

  • Accessibility from anywhere with internet
  • Automatic backups and versioning
  • Scalability (pay for what you use)
  • Collaboration features
  • Disaster recovery
