Evolution of the Internet: DNS & IP Addressing

Historical Development

1984 - The Domain Name System (DNS) was developed.
  • Computers on a network were assigned IP addresses for identification.
  • An IP address consists of 32 bits (or 4 bytes), divided into 4 octets (each between 0-255).

Why was DNS introduced?

  • Remembering numeric IP addresses was difficult.
  • DNS assigns human-readable names to IP addresses.
  • It converts IP addresses to domain names and vice versa.

IP Address Structure

IP Address is divided into:

  • Network ID (Net ID)
  • Host ID

Internet Management Authorities:

  • IANA (Internet Assigned Numbers Authority) - Assigns network IDs.
  • ISP (Internet Service Provider) - Provides internet services.
1990 - The ARPANET network was decommissioned, but its functionality continued through other networks.
1992 - The introduction of Gopher protocol and the World Wide Web (WWW).

Important Internet Developments:

  • The first WWW server and WWW client were created by Tim Berners-Lee.
  • Standardization of URL (Uniform Resource Locator), HTML (HyperText Markup Language), and HTTP (HyperText Transfer Protocol).
1993 - The first graphical internet browser, Mosaic, was released.

Backbone Network

A Backbone Network is a high-speed transmission line that connects smaller networks.

  • It transmits data in bulk at high speeds.
  • It consists of different types of networks connected together.
  • ISPs (Internet Service Providers) are clients interconnected within the backbone network.

Network Nodes & Backbone Connectivity

What are Network Nodes?

A network node is a connection point in a network used for data switching.

  • Local ISPs provide internet services to homes and businesses.
  • Larger companies connect directly to the backbone network for faster and more stable connectivity.

Network Technologies for Backbone

1. Token Ring

  • Token Ring is a Local Area Network (LAN) technology.
  • Computers are connected in a ring or star topology.
  • It uses a token-passing scheme to manage network access.
  • Collisions occur when two computers transmit data simultaneously.
  • It is the second most widely used LAN technology after Ethernet.

2. Ethernet

  • Ethernet is the most widely used LAN technology.
  • Follows the IEEE 802.3 standard, originally developed by Xerox.
  • Uses coaxial cables or twisted-pair cables.
  • Supports wireless LANs as well.
  • The 10BASE-T Ethernet standard provides speeds up to 10 Mbps.

CSMA/CD (Carrier Sense Multiple Access with Collision Detection)

  • CSMA/CD is a network protocol used in Ethernet for managing data collisions.
  • When two computers send data simultaneously, a collision occurs.
  • CSMA/CD detects collisions and ensures retransmission after a random delay.

FDDI (Fiber Distributed Data Interface)

  • FDDI is a standard for high-speed data transmission in LANs and MANs.
  • Extends the network range up to 200 km.
  • Derived from the IEEE 802.4 Token Bus protocol.
  • Supports thousands of computers in a Metropolitan Area Network (MAN).

FDDI Network Structure:

  • Primary Ring - Provides a 100 Mbps data speed.
  • Secondary Ring - Used as a backup in case of primary ring failure.
  • If the secondary ring is not used for backup, the FDDI network speed reaches 200 Mbps.

DNS Clarification:

DNS: Domain Name System (Not "Do Not Disturb")

The DNS is the phonebook of the Internet, translating domain names to IP addresses.

Optical Fiber Backbone Network

  • Modern backbone networks use fiber-optic cables for high-speed data transmission.
  • These fiber-optic links are known as Optical Carriers (OC).
  • They use electromagnetic light waves to transmit data efficiently while reducing interference.

SONET

Synchronous Optical Network Technology

  • SONET is a high-speed fiber-optic networking technology.
  • It ensures stable digital signal transmission and enhances data reliability.
  • Provides standardized protocol for communication between different vendors' equipment.

Network Speeds

Standard Ethernet Speeds:

  • 10 Mbps (Original Ethernet)
  • 100 Mbps (Fast Ethernet)
  • 1000 Mbps (Gigabit Ethernet)

FDDI Speeds:

  • Primary Ring: 100 Mbps
  • Combined Capacity: 200 Mbps
