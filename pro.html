Central Processing Unit (CPU)

The brain of the computer that performs processing operations

CPU Overview

The Central Processing Unit (CPU) is the primary component of a computer that performs most processing inside the computer. It's often called the "brain" of the computer.

CPU has three main components:
  1. Control Unit (CU) - Manages instruction execution
  2. Arithmetic Logic Unit (ALU) - Performs calculations
  3. Instruction Register (IR) - Temporary data storage

Control Unit (CU)

The Control Unit directs the operation of the processor. It acts as the central nervous system of the computer, coordinating all activities.

Main Functions:

  • Manages and coordinates all computer operations
  • Controls input/output operations
  • Manages data flow between different components
  • Performs the fetch-decode-execute cycle
Fetch-Decode Process:
  1. Fetch - Retrieves instructions from memory
  2. Decode - Interprets the instructions
  3. Execute - Carries out the instructions

Arithmetic Logic Unit (ALU)

The ALU performs all arithmetic and logical operations for the processor.

Arithmetic Operations:

  • Addition, subtraction, multiplication, division
  • Increment and decrement operations
  • Floating point calculations

Logical Operations:

  • AND, OR, NOT, XOR operations
  • Comparison operations (greater than, less than, equal to)
  • Bit shifting operations
Example: When you add two numbers (5 + 3), the ALU performs the actual calculation and returns the result (8).

Instruction Register (IR)

The Instruction Register is a temporary storage location within the CPU that holds the current instruction being executed.

Key Functions:

  • Holds the instruction currently being executed
  • Temporarily stores data during processing
  • Facilitates data transfer between CPU components
  • Works with other registers like the Program Counter (PC)
Register Types:
  1. IR - Instruction Register
  2. PC - Program Counter (points to next instruction)
  3. MAR - Memory Address Register
  4. MDR - Memory Data Register
