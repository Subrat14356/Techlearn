Computer Output Devices

Learn about the hardware components that display or produce computer results

Output Unit Overview

The output unit consists of devices that present processed data from the computer to users. Output can be categorized into two types:

Output Types:
  1. Soft Copy - Temporary output displayed on screens (monitors)
  2. Hard Copy - Permanent output printed on paper (printers)

Monitor (Soft Copy Output)

Monitors display visual output from the computer. Modern monitors come in several technologies:

Monitor Types:

  1. CRT (Cathode Ray Tube) - Older, bulky monitors using electron guns
  2. LCD (Liquid Crystal Display) - Thin, flat panels using liquid crystals
  3. LED (Light Emitting Diode) - Energy-efficient LCDs with LED backlighting
  4. TFT (Thin Film Transistor) - Active matrix LCDs with better image quality
Resolution Examples:
  • HD: 1280×720 pixels
  • Full HD: 1920×1080 pixels
  • 4K UHD: 3840×2160 pixels

Desktop Interface

The graphical user interface displayed on monitors after operating system loads:

Desktop Components:

  • Desktop - The main screen area with icons and background
  • Icons - Small graphical representations of programs/files
    Examples: Notepad, Paint, Microsoft Word, Recycle Bin
  • Taskbar - The bar at the bottom (or side) of the screen showing running applications and system tray

Printers (Hard Copy Output)

Printers produce permanent physical copies of digital documents on paper or other media.

Printer Categories:

  1. Impact Printers - Physical contact with paper
    • Uses inked ribbon and striking mechanism
    • Slower speed, lower quality, noisy operation
    • Examples: Dot Matrix, Drum, Chain printers
  2. Non-Impact Printers - No physical contact
    • Uses ink spraying or laser technology
    • Faster speed, higher quality, quiet operation
    • Examples: Inkjet, Laser printers

Printer Comparison

Feature Impact Printers Non-Impact Printers
Mechanism Physical striking Ink/Laser
Speed Slow Fast
Quality Low High
Noise Loud Quiet
